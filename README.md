# GuildGuide
A webpage dedicated to bringing you all the relevant game guides in a single place.

- As a regular user, I want to browse and discover guides relevant to my interests, save my favorite ones, and rate them so I can improve my gameplay and skills.

- As a guide creator, I want to publish guides on the platform so that I can share my knowledge with other players and gain recognition, rewards, or financial benefits.

- As an administrator, I want to manage and maintain the platform to ensure a smooth experience for users and creators while supporting the growth and monetization of the service.
