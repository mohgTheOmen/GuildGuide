# GuildGuide
A webpage dedicated to bringing you all the relevant game guides in a single place.

- As a regular user I want to use the site to find guides relevant to me, save them and give ratings in order to improve my play style and abilities.

- As a guide creator I want to create guides on the platform in order to provide other players with tutorials and earn recognition/credits/money.

- As an admin I want to administer the site in order to prove regular users and guide creators with a platform and earn money.
